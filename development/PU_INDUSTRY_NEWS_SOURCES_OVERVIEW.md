# Reliable PU Industry News Sources – Complete Overview

Reference list of trusted sources for polyurethane industry news and intelligence. Use for report sourcing, Assistant instructions, or editorial guidelines.

---

## 1. PU-specific / industry trade (primary)

| Source | Description | Coverage |
|--------|-------------|----------|
| **Urethanes Technology International (UTI)** | Audited magazine since 1984; “most authoritative source of news and technical information for the global polyurethanes industry.” Website: utech-polyurethane.com. Daily news, weekly newsletters, six print issues/year. | Global: Americas, Europe, Asia. Sectors: flexible/rigid foam, CASE, composites. End markets: automotive, construction, footwear, furniture. Regulatory, events (UTECH conferences). |
| **PU magazine** | Polyurethane-focused trade publication. | Sustainability, recycling, flexible/rigid foam, TPU, applications. Scientific and industry interviews. |
| **PUdaily** | Independent polyurethane market intelligence platform. | Prices, data, analysis for the global PU value chain. News, syndicated/custom reports, events. CBAM, bio-based PU, market dynamics. |
| **Utech-polyurethane.com** | UTI’s web presence; daily news. | Categories: financial news, sustainability, materials/cases, foam products. Regional and sector breakdowns. |

| Source | Description | Relevance to PU |

## 4. Trade bodies and regulatory

| Source | Description | What they publish |
|--------|-------------|-------------------|
| **Center for the Polyurethanes Industry (CPI)** | American Chemistry Council; North American PU industry. | Advocacy, safety, benefits messaging; annual Polyurethane Technical Conference. |
| **ISOPA** | European Diisocyanate & Polyol Producers Association. | EU PU raw materials; regulatory, safety, socio-economic studies. |
| **ALIPA** | European Aliphatic Isocyanates Producers Association. | Aliphatic isocyanates in Europe; often joint initiatives with ISOPA. |
| **International Isocyanate Institute (III)** | Global diisocyanate information. | Technical and safety information. |
| **Regional PU/chemical associations** | e.g. China, Japan, Korea, India, Brazil (links via ACC, ISOPA, diisocyanates.org). | Regional policy, events, statistics. |

Use for: regulatory updates, official positions, event calendars—not day-to-day news unless they issue press releases or reports.

## 5. Company and official sources

| Type | Description |
|------|-------------|
| **Company press releases** | Official announcements from PU producers, systems houses, foamers, equipment suppliers. |
| **Investor relations / earnings** | Listed companies (e.g. BASF, Covestro, Dow, Huntsman, Recticel, etc.). |
| **Regulatory and government** | EU (ECHA, DG GROW), US (EPA, OSHA), national agencies; trade and customs. |

Always attribute to the issuing body and use publication/first-issue date.

---

## 6. Sourcing best practices (short)

- **Diversity:** Use several categories above (PU-specific + chemical/plastics/coatings + financial + official) per report.
- **Attribution:** Every item should name the source and publication date (e.g. “— Source Name (YYYY-MM-DD)”).
- **Date:** Use first publication date only; exclude items without a verifiable date in the lookback window.
- **Credibility:** Prefer established trade and financial outlets; avoid unverified or heavily biased sources.
