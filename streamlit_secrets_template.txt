# ============================================
# WHERE TO FIND YOUR VALUES:
# ============================================
#
# SUPABASE_URL:
#   - Go to your Supabase project dashboard
#   - Settings → API
#   - Copy "Project URL"
#
# SUPABASE_ANON_KEY:
#   - Same Supabase dashboard
#   - Settings → API
#   - Copy "anon public" key
#
# OPENAI_API_KEY:
#   - Go to https://platform.openai.com/api-keys
#   - Create new key or use existing one
#   - Format: sk-...
#
# OPENAI_ASSISTANT_ID:
#   - Go to https://platform.openai.com/assistants
#   - Select your Assistant
#   - Copy the Assistant ID
#   - Format: asst_...
#
# OPENAI_VECTOR_STORE_ID:
#   - Optional, only if using company list
#   - Found in your .env file or OpenAI dashboard
#   - Format: vs_...
#
# OWNER_EMAIL:
#   - Your admin email address
#   - Used for Admin app login
#
# OWNER_PASSWORD:
#   - Your admin password
#   - Used for Admin app login
#
# ORDER_WEBHOOK_URL:
#   - Make.com webhook URL for order notifications
#   - Format: https://hook.integromat.com/...
#   - Optional: If not set, webhooks are disabled
#
# WEBHOOK_SECRET:
#   - Shared secret for webhook authentication
#   - Sent as X-Webhook-Secret header
#   - Optional: If not set, no authentication header is sent
#
# SOURCE_APP:
#   - Source application identifier
#   - Default: "pu_observatory"
#   - Alternative: "pu_ondemand_training"
#   - Optional: Defaults to "pu_observatory" if not set
